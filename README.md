# Multi-Agent Path Finding System (MAPF)

An intelligent Multi-Agent Path Finding (MAPF) system designed to simulate autonomous agent navigation in shared environments using graph-based shortest path and collision avoidance algorithms.

This project focuses on solving real-world coordination and navigation problems where multiple agents must move efficiently from source to destination while avoiding deadlocks, conflicts, and collisions.

---

## Project Overview

The system models an environment as a graph and computes optimized paths for multiple agents simultaneously. It implements classical pathfinding algorithms and coordination mechanisms to ensure safe and efficient traversal in constrained environments.

The project demonstrates concepts from:

- Artificial Intelligence
- Graph Theory
- Autonomous Systems
- Algorithm Design
- Multi-Agent Coordination
- Simulation Systems

## Algorithms Used

### Dijkstra Algorithm
Used for computing the shortest path between nodes in weighted graphs.

### A* Search Algorithm
Improves search efficiency using heuristic-based traversal.

### Conflict Resolution Logic
Handles collisions and overlapping movements between multiple agents.

## System Architecture

1. Environment is represented as a graph.
2. Agents are assigned source and destination nodes.
3. Shortest path algorithms compute optimal routes.
4. Conflict detection checks overlapping paths.
5. Collision avoidance logic recalculates movement.
6. Final optimized paths are executed in simulation.
